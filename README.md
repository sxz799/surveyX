# surveyX

一个**部署简单**的问卷调查系统

部署有多简单呢?
解压后直接运行即可,无需 数据库 nginx等环境

## 技术框架

### 后端
go gin gorm jwt sqlite mysql postgres
 
### 前端
Vue3 vite element-plus

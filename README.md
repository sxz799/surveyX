# surveyX

一个简单的问卷调查系统  
单文件运行无需配置环境,支持不同系统   
纯本地部署,保证数据安全[适合企业内部使用]

## 技术框架

### 前端
vue3 vite element-plus

### 后端
go gin gorm jwt 

### 数据库
sqlite[推荐,默认]  mysql  postgres
